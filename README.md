# fed

`fed` is a minimalist Win32 text editor built on Scintilla.

It focuses on a small, practical feature set:

- Open, edit, and save plain text files
- Line numbers and status bar
- Word wrap toggle
- Zooming
- Light and dark theme modes
- Basic file and edit commands with keyboard shortcuts


## Keyboard Shortcuts

- `Ctrl+N` - New
- `Ctrl+O` - Open
- `Ctrl+S` - Save
- `Ctrl+Shift+S` - Save As
- `Ctrl+Z` - Undo
- `Ctrl+X` - Cut
- `Ctrl+C` - Copy
- `Ctrl+V` - Paste
- `Ctrl+A` - Select All
- `Ctrl++` - Zoom In
- `Ctrl+-` - Zoom Out
- `Ctrl+0` - Reset Zoom
